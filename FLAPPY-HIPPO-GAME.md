# Flappy Hippo - Новая игра

## Описание
Flappy Hippo - это игра в стиле Flappy Bird, где бегемотик летит сквозь трубы.

## Механика игры
- **Управление**: Клик/тап по экрану заставляет бегемотика подпрыгнуть
- **Цель**: Пролететь между трубами как можно дольше
- **Очки**: +1 очко за каждую пройденную трубу
- **Проигрыш**: Столкновение с трубой или краем экрана

## Параметры игры
- **Гравитация**: 0.5 (скорость падения)
- **Сила прыжка**: -8 (скорость подъема)
- **Ширина трубы**: 80px
- **Промежуток между трубами**: 180px
- **Скорость движения труб**: 3px/кадр
- **Размер бегемотика**: 60px
- **Интервал появления труб**: 2 секунды

## Награды
- **Монеты**: 5 монет за каждое очко
- **Энергия**: -20% при начале игры
- **Счастье**: +10% при завершении игры с очками

## Файлы
- `src/components/FlappyHippo.tsx` - основной компонент игры
- `src/components/FlappyHippo.css` - стили игры
- `src/pages/GamesPage.tsx` - интеграция в список игр

## Особенности реализации
- Использует `requestAnimationFrame` для плавной анимации
- Физика движения с гравитацией и инерцией
- Поворот бегемотика в зависимости от скорости падения/подъема
- Адаптивный дизайн для мобильных устройств и landscape режима
- Процедурная генерация труб с случайной высотой промежутка

## Как играть
1. Перейдите на страницу "Игры"
2. Пролистайте до игры "Летающий бегемот"
3. Нажмите зеленую кнопку для старта
4. Кликайте по экрану, чтобы бегемотик подпрыгивал
5. Избегайте столкновений с трубами
