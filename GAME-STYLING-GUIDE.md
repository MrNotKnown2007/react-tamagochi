# Руководство по изменению стилей игр

## Как самостоятельно менять позиции элементов

### 1. Основные CSS свойства для позиционирования

**margin-top** - отступ сверху (опускает элемент вниз)
**margin-bottom** - отступ снизу (поднимает следующий элемент)

Пример:
```css
.game-over-title {
  margin: 100px 0 25px 0;
  /* Это значит: сверху 100px, справа 0, снизу 25px, слева 0 */
}
```

### 2. Адаптивные размеры с clamp()

Формат: `clamp(минимум, предпочтительный, максимум)`

```css
margin-top: clamp(80px, 20vh, 120px);
```
- **80px** - минимальный отступ (на маленьких экранах)
- **20vh** - предпочтительный (20% высоты экрана)
- **120px** - максимальный отступ (на больших экранах)

### 4. Как опустить элементы ниже

#### Опустить текст "Поражение!" в игре с шариками:

Откройте `src/components/BubbleGame.css` и найдите:
```css
.game-over-title {
  margin: clamp(120px, 30vh, 160px) 0 clamp(25px, 6.25vh, 35px) 0;
}
```

Чтобы опустить ниже, увеличьте первое число:
```css
.game-over-title {
  margin: clamp(150px, 35vh, 180px) 0 clamp(25px, 6.25vh, 35px) 0;
  /* Увеличили 120px→150px, 30vh→35vh, 160px→180px */
}
```

#### Опустить кнопки:

Найдите:
```css
.result-buttons {
  margin-top: clamp(80px, 20vh, 120px);
}
```

Увеличьте значения:
```css
.result-buttons {
  margin-top: clamp(100px, 25vh, 140px);
}
```

### 5. Адаптация для разных устройств

В файлах CSS есть секции для разных экранов:

**Десктоп (основные стили)** - без @media
**Мобильные** - `@media (max-width: 480px)`
**Горизонтальный режим** - `@media (max-height: 600px) and (orientation: landscape)`

Пример изменения для мобильных:
```css
@media (max-width: 480px) {
  .game-over-title {
    margin: clamp(100px, 25vh, 130px) 0 clamp(18px, 4.5vh, 25px) 0;
    /* Измените эти значения для мобильных */
  }
}
```

### 6. Быстрая шпаргалка

| Что нужно | Какое свойство менять | Как |
|-----------|----------------------|-----|
| Опустить элемент | `margin-top` | Увеличить значения |
| Поднять элемент | `margin-top` | Уменьшить значения |
| Увеличить расстояние между элементами | `margin-bottom` | Увеличить |
| Изменить размер | `width`, `height`, `font-size` | Изменить clamp() |

### 7. Пример: Опустить все в экране поражения

```css
/* Десктоп */
.game-over-title {
  margin: clamp(150px, 35vh, 180px) 0 clamp(30px, 7vh, 40px) 0;
}

.result-buttons {
  margin-top: clamp(100px, 25vh, 140px);
}

/* Мобильные */
@media (max-width: 480px) {
  .game-over-title {
    margin: clamp(120px, 30vh, 150px) 0 clamp(20px, 5vh, 28px) 0;
  }
  
  .result-buttons {
    margin-top: clamp(80px, 20vh, 110px);
  }
}
```

### 8. Советы

1. **Всегда меняйте все три значения в clamp()** пропорционально
2. **Тестируйте на разных экранах** - десктоп, мобильный, горизонтальный
3. **Сохраняйте файл** и обновляйте браузер (Ctrl+R или F5)
4. **Делайте изменения постепенно** - по 10-20px за раз
